PROJECT DESCRIPTION:

Our subject in school required us to create a product that we can make money from. Since then, many students have been selling physical products such as QR code keychains, stickers, and other technology-related items. However, I started thinking, what if instead of creating another physical product, we build a website where these businesses can be listed and use our system as their platform? When I shared this idea, my teacher was proud of it because our group would be the first to create something like this. Most of the other groups are still focusing mainly on physical products related to technology.
